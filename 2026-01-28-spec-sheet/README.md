# Spec Sheet

**Date:** February 2, 2026
**Type:** Typography / Type Specimen

## Overview
A comprehensive type specimen sheet with interactive font pairing selector. Features colorful pairing cards inspired by the Color Spec experiment, with full typographic showcases including size scales, weight ramps, and character sets.

## Key Features
- Interactive font pairing selector with 6 pairings
- Colorful specimen cards (purple, coral, blue, green, red, yellow)
- Size scale demonstration (72px to 14px)
- Weight ramp showing all available weights
- Full character set display with numbers and symbols
- Pull quote with drop cap
- Body text samples
- Dark/light mode toggle
- Text-based controls with underline hover animations

## Design Principles
- Typography as the primary visual element
- Color used strategically in pairing selector only
- Subtle, refined hover interactions
- Professional specimen sheet aesthetic
- Strong hierarchy through scale and weight

## How to Run
Open `index.html` in a browser or use:
```bash
npm run dev
```

## Dependencies
- Google Fonts: Fraunces, Inter, DM Serif Display, Plus Jakarta Sans, Playfair Display, Source Sans 3, Space Grotesk, Source Serif 4, Sora, Spectral, Abril Fatface, Work Sans
