# Geist Pixel

**Date:** February 6, 2026
**Type:** Typography Specimen

## Overview

A typographic specimen page exploring Geist Pixel, Vercel's bitmap-inspired typeface with 5 distinct variants. Features a black background with white text, interactive variant switching, and comprehensive character displays.

## Key Features

- **5 Pixel Variants**: Square, Grid, Circle, Triangle, Line
- **Interactive Selector**: Click to switch all specimen text between variants
- **Variant Grid**: Visual comparison cards showing each style
- **Size Scale**: 72px down to 14px readability test
- **Character Set**: Full uppercase, lowercase, numbers, symbols
- **Variant Comparison**: Side-by-side rendering of the same text

## Design Principles

- Pure black/white palette for maximum contrast
- Pixel-perfect aesthetic matching the font's digital origins
- Minimal UI letting typography take center stage
- Click interactions reveal variant differences

## How to Run

Open `index.html` in a browser. Fonts load from unpkg CDN.

## Dependencies

- Geist Pixel fonts (5 variants) via unpkg CDN
- Geist Sans for UI text via unpkg CDN
- No build step required

## Font Details

- **Source**: [vercel.com/font](https://vercel.com/font)
- **CDN**: `https://unpkg.com/geist@1.7.0/dist/fonts/geist-pixel/`
- **Glyphs**: 480 characters per variant
- **Construction**: Pixel-grid based, manually refined
